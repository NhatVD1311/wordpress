## 🛡️ Detection Patterns

### Network Level
```
GET /wp-admin/admin-ajax.php?action=backuply_backup&file=../
GET /wp-admin/admin-ajax.php?action=cartflows_debug&file=..
GET /wp-admin/admin.php?page=aruba-hispeed-cache&file=..
```

### Patterns to Alert On
```
- Multiple ../ in query/post parameters
- action= parameter combined with file= parameter
- Requests to admin-ajax.php from non-admin IPs
- Repeated 404s followed by 200 responses (path discovery)
- Large response sizes from admin-ajax.php
```

### Log Search
```bash
# Find exploitation attempts
grep -E "(cartflows_debug|backuply_backup|aruba.*file=)" /var/log/apache2/access.log | grep -E "(\.\./|%2e%2e|%252f)"

# Find sensitive file access
grep -E "(wp-config|passwd|shadow|\.env)" /var/log/apache2/access.log
```

1. Plugin configuration files
2. WordPress user table (via DB)

### Low Priority (Hard Access, Low Value)
1. System files
